Privacy Policy

UGC Lab — AI Video Ads Studio · Cập nhật: 27 tháng 5, 2026

1. Thông tin chúng tôi thu thập

UGC Lab thu thập các thông tin sau khi bạn sử dụng app:

  • Email + mật khẩu — qua Supabase Auth, để bạn đăng nhập
  • API keys bạn cung cấp (Gemini, KIE, ElevenLabs, YouTube Data API, v.v.) — lưu chỉ trong browser của bạn (localStorage) + sync sang Supabase row của bạn để dùng cross-device
  • Nội dung bạn tạo — kịch bản, avatar, voice clip, video render — lưu vào Supabase account của bạn
  • Settings ứng dụng — preferences, draft state — localStorage

2. Cách chúng tôi sử dụng YouTube Data

Tính năng "Tìm Source Video" sử dụng YouTube Data API v3 theo cách sau:
  • Khi bạn submit kịch bản → AI tách scene → app gọi search.list để tìm video YouTube Shorts liên quan
  • Kết quả hiển thị dạng thumbnail + link tới YouTube — click sẽ mở video TRÊN trang youtube.com
  • App có thể fetch transcript (auto-captions công khai) qua serverless proxy để rank kết quả tốt hơn

Chúng tôi KHÔNG:

  • Download, lưu trữ, hoặc redistribute video YouTube
  • Modify metadata YouTube
  • Truy cập data riêng tư của user YouTube (không dùng OAuth, chỉ API key)
  • Scrape hoặc harvest content YouTube ngoài mục đích discovery

3. Lưu trữ và Cache

  • Browser cache: kết quả search YouTube + Gemini parsing được cache trong localStorage 1-7 ngày để giảm chi phí API. Bạn có thể clear bằng nút "Xóa cache" trong app.
  • Supabase: data bạn tạo (avatar, script, render output) lưu vào account riêng. Bạn có quyền xóa anytime.
  • Không có server-side cache của YouTube data ngoài CDN edge cache 24h của serverless function (chỉ chứa transcript text công khai).

4. Bên thứ ba

Chúng tôi dùng các dịch vụ sau, mỗi dịch vụ có policy riêng:

  • Google Gemini API — parse kịch bản, embedding, image gen
  • YouTube Data API v3 — discovery video công khai
  • KIE.AI — image + video generation
  • ElevenLabs — text-to-speech
  • Supabase — auth + data storage
  • Vercel — hosting + serverless functions

5. YouTube API Services Terms

Khi sử dụng tính năng "Tìm Source Video", bạn đồng ý chịu thêm:

  • YouTube Terms of Service
  • Google Privacy Policy

App tuân thủ YouTube API Services — Required Minimum Functionality.

6. Quyền của bạn

  • Xem data đã lưu (qua app UI)
  • Xóa data anytime (Settings → xóa account)
  • Export data (Supabase backup)
  • Revoke API keys anytime (clear trong Settings)

7. Liên hệ

Câu hỏi về privacy → duongtrienhieu@gmail.com

Privacy Policy (English)

UGC Lab — AI Video Ads Studio · Last updated: May 27, 2026

1. Information We Collect

  • Email + password via Supabase Auth for login
  • API keys you provide — stored in your browser's localStorage and synced to your Supabase row for cross-device use
  • Content you create — scripts, avatars, voice clips, rendered videos — stored in your Supabase account
  • App settings — preferences and draft state in localStorage

2. YouTube Data Usage

The "Tìm Source Video" feature uses YouTube Data API v3:

  • Calls search.list to find publicly available short-form videos matching scene concepts
  • Displays results as thumbnails with deep-links to youtube.com
  • May fetch public auto-captions (transcripts) via serverless proxy to improve ranking

We do NOT:

  • Download, store, or redistribute YouTube content
  • Modify YouTube metadata
  • Access private YouTube user data (no OAuth — API key only)
  • Scrape or harvest content beyond discovery purposes

3. Caching

  • Search results cached in browser localStorage for 1-7 days to reduce API costs. Users can clear via the "Xóa cache" button.
  • No server-side caching beyond Vercel edge cache (24h) of public transcript text only.

4. Third Parties

We use Google Gemini API, YouTube Data API v3, KIE.AI, ElevenLabs, Supabase, and Vercel — each with their own policies linked in the Vietnamese section above.

5. YouTube API Compliance

By using the "Tìm Source Video" feature, you also agree to YouTube ToS and Google Privacy Policy. The app complies with YouTube API Services — Required Minimum Functionality.

6. Your Rights

View / delete / export your data anytime via the app. Revoke API keys via Settings.

7. Contact

Privacy questions: duongtrienhieu@gmail.com
